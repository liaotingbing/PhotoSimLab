# 基本原理 


## 简介
有限差分本征模式（FDE）方法是一种高效、精确的数值技术，广泛用于光波导与微纳光子器件的模场分析。随着集成光子学向高集成度、低损耗、多功能方向发展，精确获取波导本征模式成为设计与优化器件性能的关键。传统解析方法难以处理复杂几何或材料分布，而有限差分法将微分本征方程离散为大型稀疏矩阵特征值问题，结合完美匹配层（PML）等边界条件，可在全矢量形式下快速求解任意截面波导的模场分布、有效折射率及损耗特性。近年来，伴随计算硬件与算法并行化的进步，FDE 已被拓展至弯曲波导、各向异性材料、非线性及增益介质，为硅基光互连、激光器、传感器及量子光子芯片的研发提供了可靠的理论支撑与设计工具。
